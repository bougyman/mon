6. Le tableau des clients

Par défaut, la vue croisée liste les alias dans la première colonne mais ne détaille rien.

Pour voir le détail d'un alias, sélectionnez-le dans la colonne ``Liste'' et cliquez sur le bouton Envoyer. Vous pouvez effectuer une sélection multiple.

Pour sélectionner ou déselectionner tous les clients, vous utilisez les boutons radio ``tous'' ou ``aucun'' et cliquez sur le bouton envoyer.
