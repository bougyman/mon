7. The historic table

In red, the alerts. In green, the up-alerts and the start-alerts.

You choose the number of line you want to see with the No column text-field. You filter the last happened events with the Filter pop-up. The last alerts within the interval are displayed with a broken blank background. You can filter with the date. You can filter which services or groups you wan to see. All those criteria are used with a ``logical and''.
