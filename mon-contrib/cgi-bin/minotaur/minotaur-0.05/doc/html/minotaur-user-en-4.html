4. The preference table

With the refresh pop-up you select the time refresh time of the whole page.

With the language pop-up you select the user interface language.

With the check-boxes named Infos, Status, Historic, Cross View, you select the tables you want to see.

You can login with the User and Password text-fields. It may allow you to perform special action like enabling and disabling host or services.
